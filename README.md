## Project Overview

This project is a simple landing page for a VPN service. It is built using Tailwind CSS and the AOS library for animations.

## Technologies Used

- Tailwind CSS
- AOS library
- Swiper library
- Font Awesome icons
- Google Fonts



## Features

- Smooth animations
- Responsive design
- Easy to customize

2. Install the dependencies
